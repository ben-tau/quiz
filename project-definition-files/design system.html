Accessibilité

BrainBattle intègre des principes d'accessibilité pour assurer une expérience utilisateur inclusive.

Principes d'accessibilité
  • Contraste : Maintenir un ratio de contraste minimum de 4.5:1 pour le texte standard et 3:1 pour le texte large
  • Focus visuel : Tous les éléments interactifs doivent avoir un indicateur de focus visible
  • Alternatives textuelles : Fournir des alternatives textuelles pour tous les éléments non-textuels
  • Navigation au clavier : Assurer que toutes les fonctionnalités sont accessibles via le clavier
  • Structure sémantique : Utiliser les éléments HTML appropriés pour la structure du contenu
Recommandations :
  • Testez régulièrement l'interface avec des outils d'accessibilité
  • Assurez-vous que les animations peuvent être désactivées via prefers-reduced-motion
  • Utilisez ARIA lorsque nécessaire pour améliorer la compréhension par les technologies d'assistance
